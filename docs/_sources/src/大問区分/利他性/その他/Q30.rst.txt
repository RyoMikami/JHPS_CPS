.. rst-class:: question
問題文
==================


仮にあなたの5歳のこどもが高い熱を出して苦しんでいるとします。医者は、熱も苦痛もまったく害をもたらさないと言います。一方、この症状を治す薬を飲むと子供が大人になったときの病気に対する抵抗力が少し弱くなります。この時、あなたは子供に薬を飲ませますか？

1．高熱の症状が1日つづくと知られている場合、薬を子どもに飲ませる。

2．高熱の症状が2日つづくと知られている場合、薬を子どもに飲ませる。

3．高熱の症状が1週間つづくと知られている場合、薬を子どもに飲ませる。

4．高熱の症状が1ヶ月つづくと知られている場合、薬を子どもに飲ませる。

5．子供に薬を飲ませない。
